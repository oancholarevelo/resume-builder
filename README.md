# Dynamic Resume Builder

A simple yet powerful web-based resume builder that allows you to dynamically create, preview, and download a professional-looking resume as a PDF. This project was built with pure HTML, CSS, and JavaScript, using modern tools like Tailwind CSS for styling and jsPDF for PDF generation.

## 🚀 Features

* **Live Preview:** See your resume update in real-time as you type.
* **Dynamic Sections:** Easily add or remove multiple entries for education, work experience, skills, and more.
* **PDF Export:** Download your completed resume as a high-quality, formatted PDF with a single click.
* **Customizable Fonts:** Choose from multiple professional fonts (Modern, Classic, Typewriter) to match your style.
* **Responsive Design:** The interface is fully responsive and works smoothly on both desktop and mobile devices.
* **Justified Text:** Paragraphs like the "Objective" section are beautifully justified for a clean, professional look.
* **Pre-filled Data:** The form loads with sample data to demonstrate its functionality right away.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **PDF Generation:** [jsPDF](https://github.com/parallax/jsPDF)
* **Fonts:** [Google Fonts](https://fonts.google.com/) (Inter)
